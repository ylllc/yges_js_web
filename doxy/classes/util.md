## SafeStepIter() {#Util_SafeStepIter}

### Args

| Name | Type | Means |
|------|------|-------|
| bgn | number | start value |
| end | number | end value |
| step | number | step delta |
| cb_iter | @ref Util_CB_StepIter | loop procedure |

### Returns

next counter (same to for() counter)
