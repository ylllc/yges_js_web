# What's It?

QHT is very simple web view interface.  
1 QHT instance controls 1 HTML element.  

-----
# Import

## for web (only)

```
<script src="yges/ipl.js"></script>
<script src="yges/qht.js"></script>
```
use YgEs

-----
# How to Use

-----
## Refer an element

```
let main=YgEs.toQHT(document.getElementById('main'));
```

-----
## Create new element

```
let hw=YgEs.newQHT({
	target:main,
	tag:'div',
	attr:{class:'sub'},
	style:{border:'ridge'},
	sub:['Hello World!']
});

```

-----
## Remove an element

```
hw.Remove();
```
